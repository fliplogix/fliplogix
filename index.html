Features

  • Manage items across Unlisted, Active, and Sold
  • Group items into Lots and track quantities
  • Automatic profit calculation (fees, shipping, cost)
  • Dashboard with key KPIs and charts
  • Fast search and filtering
  • Visual alerts for missing data
  • Fully portable – no installation required

How It Works

Download the ZIP, extract it anywhere, and run the app. All your data is stored locally. To reset, simply delete the database file.

Built for simplicity, speed, and full control over your data.
