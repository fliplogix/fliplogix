# FlipLogix - an eBay flipper / reseller app.

A lightweight, portable tool to manage your inventory, lots, and sales with minimal setup. 
Everything runs from the ZIP—no installation required. Just extract, run, and start managing your items!

---

## ⚡ Features

* Add, edit, and move items between **Unlisted**, **Active**, and **Sold** states
* Assign items to **Lots** and track lot quantities
* Automatically calculate **profit**, **fees**, and other KPIs
* **Dashboard** with charts and key metrics
* Quick **search** and filtering by name, category, and dates
* **Support Us** page with Ko-fi and Amazon affiliate links
* Fully **portable**: database and media paths are relative, backup by simply copying the folder

---

## 📝 Getting Started

1. Download the **ZIP file** for the latest version.
2. Extract anywhere (USB, cloud folder, or local drive).
3. Run `FlipLogix.exe` (Windows).
4. Optional: delete `inventory.db` to reset all data and start fresh.

> ⚠️ No installation required. Just make sure all files stay together in the same folder.
